SM2综合实验
=
项目介绍
-
SM2综合项目，其中包括：  
1、ECDSA签名验证实验    
2、sm2自主实现  
3、ECMH实现  
4、PGP方案实现  
5、sm2 2P实网签名实验  
6、POC实施  
7、实网 sm2 2p解密实验

操作指导
-
关于2P通信解密和签名实验，运行时请修改注释内容  
![image](https://user-images.githubusercontent.com/95351319/182005374-74be9793-c3db-4f89-8350-408af0af3985.png)  
需要运行解密实验时使用解密函数，签名时运行签名函数。其中user1和user2两个文件需要同时修改。  
运行时，先运行user2，再运行user1  
其他文件直接运行即可
